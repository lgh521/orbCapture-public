# OrbCapture

Native screen capture and recording for macOS.

OrbCapture is available on the **Mac App Store** — no zip downloads, no manual installation required.

[Download on the Mac App Store](https://apps.apple.com/app/id6761210613) *(available once approved)*

## Features

- **Screenshots** — Full-screen, area, or window capture with global shortcuts
- **Screen recording** — Record a visible screen, a custom region, or a single app window
- **Camera bubble** — Optional floating circular camera overlay for demos and walkthroughs
- **Microphone recording** — Include your voice in recordings
- **GIF export** — Convert short recordings to animated GIFs
- **Local library** — All captures stored locally on your Mac
- **Watermark text** — Add a custom floating text watermark to recordings

## System Requirements

- macOS 14.0 Sonoma or later
